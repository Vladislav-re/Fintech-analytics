# Credit Default Risk

Скоринговая модель вероятности дефолта и расчёт оптимального порога отсечения заявок для максимизации прибыли банка.

## Решаемые задачи в рамках проекта


1. Провести анализ данных, описательную статистику данных, в т.ч. корогтный анализ по привлечению повторных кредитов (retention), vintage анализ по когортам для определения периода возникновения наибольшего кол-ва просрочек (DPD), одна когорта - количество кредитов, выданных в один и тот же помежуток времени.
2. Проверить гипотезу о том, что Default rate клиентов в официальном браке значимо ниже default rate клиентов не состоящих в официальном браке — наличие официального брака снижает вероятность дефолта
3. Предсказать вероятность дефолта (DPD90+) по кредитной заявке - построить скоринговую модель (LightGBM), сравнить ее результаты с логистической моделью предсказания дефолта.
4. Определить качество полученной модели (ROC, AUC, KS)
5. Найти порог классификации, при котором чистая прибыль банка максимальна с учётом LGD, ставки, срока кредита и издержек взыскания
6. Построить дашборд по полученным данным, в т.ч. построить графичечкую модель сравнения показателей прибыльности банка и кол-ва отказов клиентам при изменении основных исходных данных (default period, loss given default, True Default Share, Collection cost rate)

## Данные

7 связанных таблиц — заявки клиентов, кредитное бюро, история платежей, балансы карт и POS-кредитов. Только одобренные заявки (selection bias зафиксирован в ограничениях).

## Методология

**Модель** — LightGBM на ~120 признаках, агрегированных из 7 источников. Train/test split 80/20, стратификация по таргету.

**Порог** — перебор всех значений от 0 до 1, для каждого считается:
E[Net] = TN × доход_хорошего + FN_cured × доход_вылеченного - FN_true × потери_дефолт

Финансовые параметры рассчитаны из реальных данных (ставка выведена через формулу аннуитета, срок — из бюро кредитных историй).


## Ограничения

- Только одобренные заявки — поведение отказанных клиентов неизвестно
- Дефолт определён как DPD90+, не полный невозврат долга
- Момент дефолта взят как отраслевой benchmark (12 мес.), можно менять в дашборде и в нотбуке
- Ставка рассчитана косвенно через формулу аннуитета и данные бюро
